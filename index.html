AWS Mapper Guide

        
      

      

        
Getting Started

        
1. Run AWS CLI commands (listed in the left sidebar) and paste JSON output into the matching text areas

        
2. Click Render Map to visualize your infrastructure

        
3. The Account field at top labels data from different AWS accounts.

        
Keyboard Shortcuts

        

          
DToggle Design Mode

          
Shift+DCompare / Diff Mode

          
ZUndo last design change

          
EExport design plan as JSON

          
/Open resource search

          
EscClose panels / exit search

          
+/-Zoom in/out

          
Ctrl+SSave project to .awsmap file

          
Shift+AToggle Accounts panel

        

        
Save and Load

        
Save downloads your workspace as a .awsmap file. Open loads it back. Compatible between web and desktop. Auto-saved to browser every 30s.

        
Compliance

        
Click the Compliance chip in the stats bar to open the full dashboard. Checks include CIS Benchmarks, WAF rules, IAM analysis, and AWS Architecture Best Practices. Mute individual findings, filter by severity/framework, export as CSV.

        
Interactions

        
Click a VPC to lock highlight. Click a Subnet for firewall rules. Click a Gateway to highlight routes. Hover for previews.

        
Design Mode

        
Press D to enter. Use the palette to add VPCs, subnets, gateways, and resources. Press E to export a plan with CLI commands.

        
Layouts

        
Grid  VPCs side by side. Landing Zone  Hub-spoke topology. Executive  Simplified overview.
